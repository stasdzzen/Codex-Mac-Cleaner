# Alert Dialog

- Источник: официальный компонент shadcn из пресета `b1a0RuLKa` (`base-mira`).
- Примитивы: `Backdrop`, `Popup` и `Close` из `@base-ui/react/alert-dialog`.
- Сохранено: `AlertDialogMedia`, варианты кнопки подтверждения, русская подпись отмены и пообъектные подтверждения.
- Потребители: `asChild` заменён на `render`; после успешного асинхронного действия управляемый диалог закрывается явно.
- Поведение: отмена остаётся нативным `Close`; ошибка действия не закрывает диалог автоматически.
- Проверка: `pnpm --filter @codex-mac-cleaner/widget typecheck`.
