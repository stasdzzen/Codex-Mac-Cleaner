# Передача в Product-чат

Этот раздел переводит утверждённую архитектуру в ограниченный вход для следующего чата. Он не заменяет PRD, implementation prompts или GitHub Issues.

## Главный документ

* [Product input](product-input.md) — обязательные решения, ожидаемые артефакты и порядок декомпозиции.

## Граница ответственности

Архитектор поддерживает канон в `docs/`. Product-чат создаёт продуктовые документы, промпты и GitHub Issues на его основании. Изменение архитектурного решения возвращается сюда и оформляется ADR до обновления продуктового плана.
