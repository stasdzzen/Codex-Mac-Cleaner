# Codex Mac Cleaner v0.1.0-beta.1

Первый публичный beta-релиз предназначен для установки из repository
marketplace и ручной проверки владельцем на синтетических данных.

## Что вошло

* локальный Codex-плагин и MCP App Dashboard без сетевого транспорта и
  телеметрии;
* read-only профиль `application_remnants` для пользовательской Library;
* server-owned evidence, риск, blocking reason и честные coverage gaps;
* отдельные кнопки для одного объекта: оставить, исключить, переместить в
  карантин, восстановить и удалить навсегда;
* fail-closed denylist для состояния Codex, текущего проекта, credentials,
  browser profiles, personal data и локальных Git-проектов;
* deterministic package, SHA-256, CycloneDX SBOM, provenance и лицензии
  зависимостей;
* security, privacy, policy, quarantine, UI, plugin и clean-room gates.

## Установка

Требуются macOS 26+, Apple Silicon `arm64`, Node.js `24.18.x` и Codex с
поддержкой плагинов.

```bash
codex plugin marketplace add stasdzzen/Codex-Mac-Cleaner --ref v0.1.0-beta.1
codex plugin add codex-mac-cleaner@codex-mac-cleaner
```

После установки откройте новую задачу Codex и попросите проверить остатки
удалённых приложений с открытием Audit Dashboard.

## Известные ограничения beta

* ручной Real-Mac smoke владельца ещё не выполнен; сначала используйте только
  синтетические данные;
* автоматически очищать разрешено только доказуемо связанные private cache/log
  remnants; остальные категории остаются inspect-only;
* автоматического расписания, bulk actions, `/Library`, `sudo`, privileged
  helper, Homebrew upgrades и глобального управления SDK нет;
* logical bytes, quarantine bytes и наблюдаемое состояние диска показываются
  отдельно и не обещают точный прирост свободного места APFS;
* публикации в общей Plugin Directory пока нет.

## Проверка целостности

GitHub prerelease публикует tar-архив и файл SHA-256. В архив включены
`release-evidence/sbom.cdx.json` и `release-evidence/provenance.json`, связанные
с commit релизного тега. Ручной протокол остаётся незаполненным до фактической
проверки владельцем.
