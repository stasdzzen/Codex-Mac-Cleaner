# Codex Mac Cleaner v0.1.0-beta.3

Beta.3 показывает Audit Dashboard сразу после запуска проверки и обновляет в
нём server-owned прогресс. Пользователю не нужно ждать окончания аудита в
текстовом чате или копировать команды в терминал.

## Что изменилось

* Dashboard v2 открывается сразу после `audit_start` и показывает безопасную
  фазу, число обработанных кандидатов, отмену и terminal failure.
* До появления immutable revision live Dashboard не содержит findings или
  кнопок изменения. Даже после завершения запрос `revision: null` остаётся
  неактивным; действия появляются только у отдельно запрошенной integer
  revision.
* Installed applications, processes, open files и startup targets собираются
  один раз на фазу и переиспользуются между кандидатами. Package/app queries
  имеют ограниченную параллельность.
* Весь read-only аудит ограничен server deadline в пять минут. Превышение
  завершается как `AUDIT_TIMEOUT`; явная отмена пользователя остаётся отдельным
  состоянием `cancelled`.
* Packaged runtime, Dashboard v2 и clean-room tests синхронизированы с исходным
  кодом.

## Обновление с beta.2

Marketplace закреплён на точном Git-теге и не обновляется автоматически.
Удалите старую установку и source, затем добавьте beta.3:

```bash
codex plugin remove codex-mac-cleaner@codex-mac-cleaner
codex plugin marketplace remove codex-mac-cleaner
codex plugin marketplace add stasdzzen/Codex-Mac-Cleaner --ref v0.1.0-beta.3
codex plugin add codex-mac-cleaner@codex-mac-cleaner
```

После обновления полностью завершите Codex, откройте его снова и создайте
новую задачу. Попросите проверить остатки удалённых приложений и открыть Audit
Dashboard.

## Ограничения beta

* Ручной Real-Mac smoke владельца для этого release SHA ещё не завершён;
  beta.3 выпускается для этой проверки и пока не считается готовой к работе с
  ценными данными.
* В v0.1 нет автоматического расписания, массовых действий, `/Library`, `sudo`,
  privileged helper, Homebrew upgrades и глобального управления SDK.
* Actionable могут быть только доказуемо связанные private cache/log remnants;
  остальные категории остаются inspect-only.
* Публикации в общей Plugin Directory пока нет.

## Целостность артефакта

GitHub prerelease содержит deterministic tar-архив, SHA-256 и file checksums.
CycloneDX SBOM, provenance и third-party notices связаны с commit release-тега.
Ручной smoke фиксируется отдельно и не объявляется выполненным в этих заметках.
