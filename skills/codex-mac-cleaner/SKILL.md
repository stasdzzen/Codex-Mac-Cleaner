# Codex Mac Cleaner

Используй этот навык только для read-only аудита остатков приложений.

1. Кратко объясни, что проверка локальна, не использует сеть или телеметрию и не обещает найти всё при недоступных областях.
2. Вызови `audit_start` только с profile `application_remnants` и новым `requestId`.
3. Получай прогресс через `audit_status`. Если есть coverage gaps, покажи их как непроверенные области.
4. После появления завершённой actionable revision вызови `audit_results`, затем автоматически вызови `dashboard_open` для этой revision.
5. Вызывай `audit_cancel` только после явного запроса пользователя отменить текущий аудит.

Не вызывай app-only инструменты очистки, карантина или пользовательских исключений. Не формируй команды терминала и не проси текстового подтверждения шага. Любое изменение начинается только с app-visible клика пользователя и отдельного подтверждения одного объекта в Dashboard. Для `unsupported_manual` объясни границу поддержки без инструкций обхода разрешений или ручного изменения системы.
